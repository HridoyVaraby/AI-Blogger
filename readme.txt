== Description ==
Generate SEO-friendly blog posts using AI via Groq API. Automate content creation with customizable templates and AI-powered suggestions.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ai-blogger` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your API keys in Settings â†’ AI Blogger

== Frequently Asked Questions ==
= How do I get an API key? =
Visit Groq's website to create an account and get your API key.

= Can I customize the content templates? =
Yes! The plugin provides filters to modify the AI prompts and output formatting.

== External services ==

This plugin connects to the Groq API to generate blog posts using AI.
It sends the blog title and context as a request to the API and receives AI-generated content.

Service Name: Groq API
Data Sent: Blog title, user prompt
Data Received: AI-generated blog post
Privacy Policy: Groq Privacy Policy
Terms of Service: Groq Terms of Use
