== Description ==
Generate SEO-friendly blog posts using AI via Groq API. Automate content creation with customizable templates and AI-powered suggestions.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ai-blogger` directory
2. Activate the plugin through the 'Plugins' screen in WordPress

= How do I get an API key? =
Visit Groq's website to create an account and get your API key.

= Can I customize the content templates? =
Yes! The plugin provides filters to modify the AI prompts and output formatting.
